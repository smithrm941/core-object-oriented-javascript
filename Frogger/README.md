Frogger game - Michelangelo vs. Two members of foot clan

-To play, open the index.html file in a browser
-Use the arrow keys to move Michelangelo's head
-If Michelangelo crashes into one of the foot clan, you have to start over
-If Michelangelo makes it across the board to the water, you win! And you start over so you can play again and again because this game is the best.
